# 🧮 Simple Calculator

A clean, responsive, and functional web-based calculator built with **HTML5, CSS3, and JavaScript**. This project features a modern UI with smooth animations and supports basic arithmetic operations.

## 🚀 Features
*   **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
*   **Responsive Design:** Works perfectly on desktops, tablets, and mobile devices.
*   **Interactive UI:** Hover effects and button animations for a better user experience.
*   **Clear Functionality:** Easily reset the screen with a "C" or "AC" button.

## 🛠️ Technologies Used
*   **HTML5:** For the structure and layout.
*   **CSS3:** For styling, grid layout, and animations.
*   **JavaScript:** For the calculation logic and DOM manipulation.


## 📥 How to Run Locally
1.  **Clone the repository:**
    ```bash
    git clone https://github.com
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd calculator
    ```
3.  **Open the project:**
    Simply open the `index.html` file in your favorite web browser.

## 📝 How to Use
1.  Type numbers using the buttons or your keyboard (if supported).
2.  Select an operator (+, -, *, /).
3.  Press the **"="** button to see the result.
4.  Use the **"C"** button to clear the current input.
